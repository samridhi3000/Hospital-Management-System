## ğŸ§  System Working (Flow Explanation)

1. The application starts by displaying a **main menu**.
2. The user selects an option:

   * Patient Management
   * Doctor Management
   * Appointment Scheduling
   * View Records
   * Exit
3. Based on the selection, the system performs the required operation.
4. After completion, control returns to the main menu.
5. The program exits safely when the user chooses the exit option.
